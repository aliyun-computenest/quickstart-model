🚀 Qwen3 大语言模型

新一代智能对话模型 - 思考与非思考模式无缝切换

🎯 产品简介

Qwen3 是 Qwen 系列最新一代的大语言模型，提供了一系列密集（Dense）和混合专家（MOE）模型。基于广泛的训练，Qwen3 在推理、指令跟随、代理能力和多语言支持方面取得了突破性的进展。
💡 核心创新
Qwen3 首次实现了思考模式与非思考模式的智能切换，在保证推理准确性的同时兼顾响应效率，为不同应用场景提供最优解决方案。

✨ 核心特性

🧠 双模式智能切换
独特支持在思考模式（用于复杂逻辑推理、数学和编码）和非思考模式（用于高效通用对话）之间无缝切换，确保在各种场景下的最佳性能。
⚡ 超越前代推理能力
显著增强的推理能力，在数学、代码生成和常识逻辑推理方面超越了之前的 QwQ（在思考模式下）和 Qwen2.5 指令模型（在非思考模式下）。
🎭 卓越人类偏好对齐
在创意写作、角色扮演、多轮对话和指令跟随方面表现出色，提供更自然、更吸引人和更具沉浸感的对话体验。
🤖 领先 Agent 能力
可以在思考和非思考模式下精确集成外部工具，在复杂的基于代理的任务中在开源模型中表现领先。
🌍 强大多语言支持
支持 100 多种语言和方言，具有强大的多语言理解、推理、指令跟随和生成能力。

🔌 API 调用方式

🖥️ Curl 命令调用

📋 参数说明
• ${ServerIP}：内网地址或公网地址中的 IP 地址
• ${ApiKey}：页面提供的 ApiKey
• ${ModelName}：模型名称
Curl 命令调用可以直接使用服务实例概览页面中的 API 调用示例，调用模型 API 的具体结构如下： 
curl -X Post http://${ServerIP}:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${ApiKey}" \
  -d '{
    "model": "${ModelName}",
    "messages": [
      {
        "role": "user",
        "content": "给闺女写一份来自未来2035的信，同时告诉她要好好学习科技，做科技的主人，推动科技，经济发展；她现在是3年级"
      }
    ]
  }'

🐍 Python SDK 调用

⚙️ 配置说明
• ${ApiKey}：填写页面上的 ApiKey
• ${ServerUrl}：填写页面上的公网地址或内网地址，需要带上 /v1
以下为 Python 示例代码： 
from openai import OpenAI

##### API 配置 #####
openai_api_key = "${ApiKey}"
openai_api_base = "${ServerUrl}"

client = OpenAI(
    api_key=openai_api_key,
    base_url=openai_api_base,
)

models = client.models.list()
model = models.data[0].id
print(model)


def main():
    stream = True

    chat_completion = client.chat.completions.create(
        messages=[
            {
                "role": "user",
                "content": [
                    {
                        "type": "text",
                        "text": "你好，介绍一下你自己，越详细越好。",
                    }
                ],
            }
        ],
        model=model,
        max_completion_tokens=1024,
        stream=stream,
    )

    if stream:
        for chunk in chat_completion:
            print(chunk.choices[0].delta.content, end="")
    else:
        result = chat_completion.choices[0].message.content
        print(result)


if __name__ == "__main__":
    main()
